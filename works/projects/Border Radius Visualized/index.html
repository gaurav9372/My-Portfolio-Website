What is border-radius ?

The border-radius is a CSS property which rounds the corners of an element’s outer border edge. You can set a single radius to make circular corners, or two radii to make elliptical corners.

Now , lets try understanding it.

Here is a div with height 150px and width 150px

150x150
div{
height: 150px ;
width : 150px;
background: #00FFA8;
}

We can easily make its corners rounded by applying the border-radius css property. Lets have rounded corners of 20px.

150x150
div{
height: 150px ;
width : 150px;

border-radius: 20px;

background: #00FFA8;
}

Easy , isnt it ? But now the quetions is...

How border-radius works ?

Well, we know that css follows Box Model, which means everything in css has a box around it. Even when we can not see it. (You can always check its border with the border css property). When we set border-radius property to an element , Its all four corners are rounded with reference to the four assigned invisible circles with radius equal to the value which we put in border-radius property.

Lets see it in working. Hers’s another div with height and width equal to 200px. See how border-radius rounds the corners of div with reference to four circles.So the value which we write in border-radius property is actually the radius of these circles(Blue line) which guide the roundenss.

200x200

border-radius: {{roundness50}}px;

0

50

We can now see how the border-radius applies the rounded corders to any element. We used px (pixels) for applying value. We can also write the value in em , rem and %. (px and % are usually preffered over others)

What if border-radius value is equal to half the size of element ?

Lets check it out. Slide the pointer all the way to 100, that is half of 200, the width of our div element.

200x200

border-radius: {{roundness100}}px;

0

100

We are getting a circle. so now we know that to make a circle we are supposed to have a square element and set its border radius equal to half its width or we can simply write border-radius : 50% if we dont want to calculate half width of it.

What if border-radius value is equal to the size of element ?

Slide the pointer to 200px.

200x200

border-radius: {{roundness200}}px;

0

200

Now you would be thinking why the div’s border is not changing after 100px ? Should not it be like this :

Yes right , but if we look closely , we see that this new shape has width and height less than our div’s orignal height and width.

In our earlier cases when border-radius was less than or equal to half the width of element , the changed shape of our element always had dimension of 200x200 retained.


Increasing border-radius past 100 would shrink the shape and that would lead to layout disturbance in our page , thats why CSS does not rounds border of any element after element's half size.
