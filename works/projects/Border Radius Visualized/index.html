What is border-radius ?

The border-radius is a CSS property which rounds the corners of an element’s outer border edge. You can set a single radius to make circular corners, or two radii to make elliptical corners.

Now , lets try understanding it.

Here is a div with height 150px and width 150px

150x150
div{
height: 150px ;
width : 150px;
background: #00FFA8;
}

we can easily make its corners rounded by applying the border-radius css property. Lets have rounded corners of 20px.

150x150
div{
height: 150px ;
width : 150px;

border-radius: 20px;

background: #00FFA8;
}

easy , isnt it ? But now the real quetions is...

How border-radius works ?

Well, we know that css follows Box Model , which means everything in css has a box around it. Even when we can not see it. (You can always check its border with the border css property). When we set border-radius property to an element , Its all four corners are rounded with reference to the four assigned invisible circles with radius equal to the value or values which we put in border-radius property.

Lets see it in working. Hers’s another div with height and width equal to 200px. See how border-radius rounds the corners of div with reference to four circles.So the value which we write in border-radius property is actually the radius of these circles which guide the roundenss.

border-radius: {{roundness50}}px;

0

50

We can now see how the border-radius applies the rounded corders to any element. We used px (pixels) for applying value. We can also write the value in em , rem and %. (px and % are usually preffered over others)

What if border-radius value is equal to half the size of element ?

Lets check it out. Slide the pointer all the way to 100, that is half of 200, the width of our div element.

border-radius: {{roundness100}}px;

0

100

We are getting a circle. so now we know that to make a circle are supposed to have a square element and set its border radius equal to half the width of it or we can simply write bordr-radius : 50% if we dont want to calculate the half wodth of any element.
